# P27 - [DOM] Xac dinh state va dieu kien re-render

- State tam thoi nam trong gia tri input cua `form`.

- Nut `Tao CV`, `Tai du lieu bien`, `Xoa nhanh` deu goi lai `refreshPreview()`.
