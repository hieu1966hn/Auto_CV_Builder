# P12 - Thu tu doc code AutoCV cho nguoi moi

## Thu tu de hieu nhanh
1. `index.html`: xem cau truc trang, id/class, noi gan script.
2. `scripts/scripts.js`: tim event, data flow, ham cap nhat preview.
3. `styles.css`: doc bo cuc va responsive.

## Cach doc trong moi file
- Bat dau tu diem vao (button, form, event listener).
- Theo bien du lieu di den dau.
- Danh dau ham xu ly quan trong.
- Kiem tra dieu kien loi/empty state.
