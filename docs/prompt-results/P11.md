# P11 - 6 khai niem co ban qua vi du AutoCV

## 1) Input
Du lieu nguoi dung nhap vao form: ho ten, email, ky nang, kinh nghiem.

## 2) State
Trang thai du lieu hien tai cua form/preview o moi thoi diem.

## 3) Data binding
Noi input va preview voi nhau: doi input thi preview doi theo.

## 4) Render
Qua trinh cap nhat UI de hien state moi len man hinh.

## 5) Event
Su kien nhu `input`, `click` kich hoat logic cap nhat.

## 6) Validate
Kiem tra du lieu hop le truoc khi chap nhan/xuat CV.

## Cach nho nhanh
Input -> Event -> Update state -> Render preview -> Validate.
