# P07 - 15 loi thuong gap khi lam web tao CV

1. Script khong load do sai duong dan file JS.
2. Form nhap duoc nhung preview khong cap nhat.
3. Tach ky nang sai khi co khoang trang thua.
4. Kinh nghiem da dong hien thi chung 1 dong.
5. Validate email khong dung mau.
6. Bo trong truong bat buoc nhung khong bao loi.
7. UI vo bo cuc tren man hinh mobile.
8. Font tieng Viet bi loi khi export.
9. Nut tao CV bam khong co phan hoi.
10. Re-render qua nhieu gay giat UI.
11. Du lieu cu con sot lai sau reset.
12. CSS de mau chu/nen kho doc.
13. Copy-paste text co ky tu dac biet lam vo layout.
14. Du lieu preview khac voi du lieu form.
15. Sau khi sua bug A lai lam hong tinh nang B.
