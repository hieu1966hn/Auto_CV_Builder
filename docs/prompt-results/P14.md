# P14 - Vi du bug "preview khong cap nhat" va cach lan code

## Dau hieu
Nguoi dung go input nhung khu vuc preview dung yen.

## Cach lan tim nguyen nhan
1. Kiem tra script co load dung duong dan khong.
2. Kiem tra da gan `addEventListener('input', ...)` cho form chua.
3. Dat log trong ham cap nhat preview xem co duoc goi khong.
4. Kiem tra selector id co dung voi HTML khong.
5. Kiem tra ham cap nhat co gan `textContent` dung phan tu khong.

## Nguyen nhan thuong gap
- Sai duong dan script.
- Sai id/selector.
- Quen gan event listener.
- Ham update bi return som.

## Cach test lai
Nhap lien tuc 2-3 truong, preview phai cap nhat ngay moi lan go.
