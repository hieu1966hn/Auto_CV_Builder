# P03 - Luong du lieu bang chu

## Luong chinh
1. Nguoi dung nhap du lieu trong form.
2. Event input duoc kich hoat.
3. App doc du lieu tu form.
4. App xu ly text co ban (trim, tach danh sach).
5. Du lieu duoc dua vao state tam thoi.
6. Khoi preview lay state va render CV.
7. Nguoi dung tiep tuc sua -> quay lai buoc 1.

## Luong xuat PDF (muc tieu sau)
1. Nguoi dung bam nut xuat PDF.
2. App lay du lieu da render.
3. App ap dung format A4.
4. Tao file PDF de tai xuong.

## Luong bao mat (neu co backend)
1. Kiem tra can luu server hay khong.
2. Neu co: ma hoa/kiem soat truoc khi gui.
3. Neu khong: giu local va xoa sau phien.
